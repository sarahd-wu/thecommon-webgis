About This Portfolio

This portfolio is a collaboration with The Common, a literary magazine that publishes stories rooted in place. The projects here are inspired by the magazine’s dispatches — short, powerful reflections from around the world. Each project is an attempt to translate these narrative moments into digital, visual, and spatial expressions, exploring what it means to locate meaning across landscapes.

My mission is to explore place-based storytelling in our dispatches. I aim to deepen spatial understanding and invite community-centered feedback, carrying forward a dispatch's spirit of locating ourselves not just on a map, but within lived experience.

Project One

Project One

This project visualizes The Common’s dispatches—stories rooted in a strong sense of place. Readers can scroll through an interactive map to explore each dispatch, view author details, and read excerpts from the pieces.

ArcGIS Experience Builder
Features
  • Displays an interactive map with author details and excerpts.
  • Allows users to explore stories by location and gain a sense of place.
Project Two

Project Two

This project analyzes dispatch data by year of publication. A slider allows users to view dispatches by date, while hovering over the map reveals details about each country featured in the stories.

Mapbox
Features
  • Filters dispatches by year of publication.
  • Provides insights on dispatch distribution with interactive map hover details.
Project Three

Project Three

This project gathers reader feedback through a survey. Readers can rate and comment on each dispatch, share their reading experience, and choose to subscribe to The Common’s newsletter.

ArcGIS Survey123
Features
  • Collects ratings and comments from readers on each dispatch.
  • Offers an option to subscribe to The Common’s newsletter.
